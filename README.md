# Mini Projeto 3 — Análise Exploratória de Dados COVID-19

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da disciplina de Tópicos Especiais I, com o objetivo de aplicar conceitos de análise exploratória de dados utilizando Python no Google Colab.

A análise foi realizada a partir de um dataset público relacionado à vacinação contra a COVID-19, contendo informações sobre pacientes vacinados, fabricantes das vacinas, municípios e datas de aplicação.

---

# 🎯 Objetivos do Projeto

O projeto teve como principais objetivos:

- Aplicar técnicas de estatística básica;
- Realizar visualização de dados;
- Desenvolver análises temporais;
- Identificar padrões e outliers;
- Criar métricas e KPIs;
- Produzir interpretações a partir dos dados analisados.

---

# 🛠️ Ferramentas Utilizadas

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

# 📂 Dataset Utilizado

O dataset utilizado contém informações relacionadas à vacinação contra a COVID-19, incluindo:

- Idade dos pacientes;
- Sexo biológico;
- Fabricante da vacina;
- Município de aplicação;
- Data da vacinação;
- Descrição da dose aplicada.

---

# 🧹 Tratamento dos Dados

Durante o projeto foram realizados diversos procedimentos de limpeza e preparação dos dados, incluindo:

- Remoção de registros duplicados;
- Tratamento de valores ausentes;
- Conversão e padronização de datas;
- Remoção de idades inválidas;
- Padronização de nomes de municípios;
- Correção de inconsistências relacionadas aos fabricantes das vacinas.

---

# 📊 Análises Realizadas

## Estatística Descritiva
Foram calculadas medidas estatísticas como:

- Média;
- Mediana;
- Moda;
- Variância;
- Desvio padrão;
- Amplitude.

---

## Visualização de Dados

O projeto contém diferentes gráficos para análise dos dados:

- Histograma;
- Boxplot;
- Scatterplot;
- Gráfico temporal;
- Heatmap de correlação;
- Gráfico de fabricantes.

---

# 🔍 Principais Descobertas

Durante a análise foi possível identificar:

- Predominância de pacientes adultos no dataset;
- Priorização de idosos nos períodos iniciais da vacinação;
- Forte presença da vacina da Pfizer;
- Correlação negativa entre idade e período de vacinação;
- Baixa quantidade de outliers.

---

# 📈 KPIs Criados

Foram desenvolvidos indicadores importantes, como:

- Total de vacinas aplicadas;
- Média de idade dos pacientes;
- Fabricante mais utilizado;
- Mês com maior quantidade de vacinações.

---

# 🚨 Identificação de Outliers

A detecção de outliers foi realizada utilizando o método IQR (Intervalo Interquartil).

Os valores extremos encontrados estavam relacionados a pacientes com idades muito elevadas, porém plausíveis em uma base nacional de vacinação.
